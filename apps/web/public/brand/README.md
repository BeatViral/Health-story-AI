# HealthStory AI Logo Package

This folder contains the source logo assets and generated app/icon exports.

## Source

- `logo-mark.svg` - primary app mark
- `logo-horizontal.svg` - horizontal wordmark
- `logo-stacked.svg` - stacked lockup
- `favicon.svg` - simplified small-size favicon

## Generated

Generated PNG exports live in `generated/` and cover favicon, PWA, Apple touch icon, extension, and social preview sizes.

Sizes included:

- Favicons: 16, 32, 48, 64
- App/PWA icons: 128, 180, 192, 256, 384, 512
- Apple touch icon: 180
- Open Graph/social preview: 1200 x 630
